1. LINE
*** TITEL

2. LINE (8F10.5)
NKO:    NUMBER OF NORMAL MODES (<6)
NCV:    NUMBER OF POTENTIAL PARAMETERS
NEIG:   NUMBER OF PRINTED EIGEN VALUES
NKOART: COORD TYPE (0) INT COORDS (STRETCH ONLY) (1) NORMAL COORDS
NXIJ:   SWITCH FOR OFFDIAG ANHARMONICITY (ONLY TESTED FOR INT COORDS...) (0) OFF (1) ON 
NDRU:   PRINT OPTIONS (0):  STANDARD REDUCED (1): DETAILED (BASIS, INTEGRATION POINTS, POTENTIAL IN INTEGRATION) (>1): FOR DEBUGGING
        WITH NDRU=1 AND FOR NFU.LE.15 THE FULL HAMILTONIAN MATRIX IS PRINTED
        WITH NDUR>1 FULL HAMILTONIAN IS ALWAYS PRINTED
EPS:    EIGENVECTOR PRINT THRESHOLD (COEFF**2>EPS WILL BE PRINTED WITH RESPECTIVE BASIS FUNCTION)

3. LINE (5F10.5)
NFU:    NUMBER OF BASIS FUNCTIONS PER COORD    (NKO ENTRIES F10.8)

4. LINE (5F10.5)
MINT:   NUMBER OF INTEGRATION POINTS PER COORD (NKO ENTRIES F10.8) 
        --> IF 0 DETERMINE "EXACT" VALUE FROM POT AND NFU AND GAUSS-HERMITE RULES
               M(I)=(NFU(I)-1)+0.5*MAX(IQ(I))+1)
            WHERE (NFU(I)-1):  HIGHEST ORDER BASIS FUNCTION IN COORD I
                  (MAX(IQ(I)): HIGHEST ORDER POLYNOMIAL ORDER IN POTENTIAL W.R.T. COORD I

4. LINE (I2)
ND:     NUMBER OF DECIMAL DIGITS IN POTENTIAL PARAMETERS

5. LINE AND FOLLOWING
POT:    POTENTIAL IN ATOMIC UNITS
        - ONE COEFFICIENT PER LINE, NCV LINES
        - FORMAT: F(ND+2).ND,NKO*I2
          e.g. ND=9,NKO=2 "-.123456789 2 1"
